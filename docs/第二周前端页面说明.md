# 第二周前端页面说明

## 1. 首页 index.html

页面路径：

```text
frontend/index.html
backend/src/main/resources/static/index.html
```

功能说明：

- 展示系统名称“校园生活百事通”。
- 提供校园生活问题搜索框。
- 用户输入问题后，跳转到 `result.html` 查询结果页。
- 展示热门问题入口，如食堂、图书馆、宿舍报修、校园卡、打印、社团等。
- 提供“补充问题”和“管理员登录”的导航入口。

## 2. 查询结果页 result.html

页面路径：

```text
frontend/result.html
backend/src/main/resources/static/result.html
```

功能说明：

- 从 URL 参数 `q` 中读取用户输入的问题。
- 页面加载后调用后端接口：

```text
GET /api/qa/search?question=xxx
```

- 将后端返回的 FAQ 问题、答案、分类、来源、查询次数展示到页面中。
- 如果后端返回 `found=false`，页面显示“暂未找到相关答案，请尝试换个关键词或提交问答建议。”。
- 未找到答案时，页面提供跳转到 `contribute.html` 的入口。
- 如果后端服务不可用，页面提示“后端服务暂不可用，请确认 Spring Boot 已启动。”。

## 3. 用户贡献页 contribute.html

页面路径：

```text
frontend/contribute.html
backend/src/main/resources/static/contribute.html
```

功能说明：

- 提供问题、建议答案、分类三个输入字段。
- 点击“提交，等待管理员审核”按钮时，不刷新页面。
- 如果问题为空，页面提示“请输入问题”。
- 使用原生 `fetch` 调用后端接口：

```text
POST /api/contribution/add
```

- 提交成功后，页面显示“提交成功，等待管理员审核”，并清空表单。
- 如果请求失败，页面提示“提交失败，请确认后端服务已启动。”。

## 4. 管理员后台 admin.html

页面路径：

```text
frontend/admin.html
backend/src/main/resources/static/admin.html
```

功能说明：

- 页面加载后调用后端接口：

```text
GET /api/faq/list
```

- 将 FAQ 列表渲染到后台表格中。
- 表格列包括：问题、分类、来源、查询次数、操作。
- 操作列中的“编辑”“删除”按钮目前为静态按钮，暂未实现真实功能。
- 如果 FAQ 列表为空，页面显示“暂无 FAQ 数据”。
- 如果接口请求失败，页面显示“FAQ 数据加载失败，请确认后端服务已启动。”。
- 保留贡献审核、知识导入、热门统计入口的静态展示。

## 5. 管理员登录页 login.html

页面路径：

```text
frontend/login.html
backend/src/main/resources/static/login.html
```

当前说明：

- 管理员登录页目前仍为静态原型。
- 页面包含账号输入框、密码输入框和登录按钮。
- 当前没有接入真实登录接口。
- 当前没有实现管理员身份校验、会话管理或权限控制。
- 登录按钮当前用于跳转到 `admin.html`，仅支持页面演示。
